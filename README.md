# ExpoScan — ISTE 서비스 카탈로그

전시에서 본 회사·서비스를 **이미지·URL·메모만 넣으면 AI가 요약**해 카드로 정리하고, 직원들과 공유·탐색하는 도구입니다. 설치·서버 없이 **HTML 파일 하나**로 동작합니다.

## 시작하기

1. **`index.html` 파일을 더블클릭**해서 브라우저로 엽니다. (끝!)
2. 우측 상단 **⚙️ 설정**에서:
   - **내 이름** — 카드에 '추가한 사람'으로 표시됩니다.
   - **Claude API 키** — AI 자동 요약에 사용됩니다. ([키 발급](https://console.anthropic.com/settings/keys))
   - **AI 모델** — Sonnet 4.6(권장) / Haiku 4.5(빠름·저렴) / Opus 4.8(최고 품질)
3. **＋ 서비스 추가** → 이미지/URL/메모 입력 → **AI 분석** → 저장

> API 키는 **이 브라우저에만** 저장되며, Claude API 호출 시에만 사용됩니다. 외부로 전송·수집되지 않습니다.

## 직원들과 공유하기

데이터는 각자 브라우저에 저장됩니다. 함께 보려면:

- **내보내기(⬆)** — 내 카탈로그를 JSON 파일로 저장 → 동료에게 전달
- **가져오기(⬇)** — 받은 JSON을 불러오면 내 목록과 **자동 병합**됩니다.
  - 같은 항목은 최신 버전으로 갱신, 새 항목은 추가됩니다.

여러 명의 JSON을 한 명이 모아 가져오면 **통합 카탈로그**가 만들어집니다. 그 통합본을 다시 내보내 모두에게 배포하면 됩니다.

## 주요 기능

- 🃏 **카드 그리드** — 카테고리 배지, 태그, 추가한 사람, 날짜 한눈에
- 🔍 **검색 + 태그 필터** — 회사명·요약·메모·태그까지 통합 검색
- 🤖 **AI 분석** — 이미지(부스 사진/제품 화면)까지 읽어 요약·핵심 포인트·태그·카테고리 자동 생성
- 📄 **상세 보기** — 요약, 핵심 포인트, 태그, 원본 메모, 사이트 링크
- ✏️ **편집/삭제**, 🖼️ 이미지 자동 압축 저장
