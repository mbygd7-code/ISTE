# ExpoScan — ISTE 서비스 카탈로그

전시에서 본 회사·서비스를 **이미지·URL·메모만 넣으면 AI가 요약**해 카드로 정리하고, 직원들과 **실시간으로 공유**하는 도구입니다.

🌐 **라이브 사이트: https://iste-zeta.vercel.app**

## 시작하기

1. **링크 접속**(또는 `index.html` 더블클릭) — 모든 직원이 같은 목록을 봅니다.
2. 우측 상단 **⚙️ 설정**에서 **내 이름**(카드에 '추가한 사람'으로 표시)과 **AI 모델**을 고릅니다.
3. **＋ 서비스 추가** → 이미지/URL/메모 입력 → **AI 분석** → 저장

> 🤖 AI 요약은 **공용 키로 서버에서 처리**되므로 직원들은 **API 키를 입력할 필요가 없습니다.** 바로 사용하세요.

## 직원들과 공유하기 — 실시간 공용 카탈로그

서비스 데이터는 **Supabase 공용 DB**에 저장됩니다.

- 누구든 **링크만 접속하면 같은 목록**을 봅니다.
- 한 명이 서비스를 추가/수정/삭제하면 **다른 사람 화면에 실시간 반영**됩니다 (새로고침 불필요).
- 내보내기(⬆)/가져오기(⬇) JSON도 그대로 지원 — 백업이나 일괄 등록에 사용하세요. (가져오면 공용 목록에 반영)

> ⚠️ 현재는 **링크를 아는 사람은 누구나 읽기/쓰기/삭제** 가능한 사내용 설정입니다. 편집 비밀번호나 로그인으로 잠그려면 확장 가능합니다.

## 주요 기능

- 🃏 **카드 그리드** — 카테고리 배지, 태그, 추가한 사람, 날짜 한눈에
- 🔍 **검색 + 태그 필터** — 회사명·요약·메모·태그까지 통합 검색
- 🤖 **AI 분석** — 여러 장의 이미지(부스 사진·로고·간판·브로슈어)를 읽어 **회사명·웹사이트 URL·메모·요약·태그·카테고리를 자동으로 채움** (빈 칸만 채우고 모두 수정 가능)
- 🖼️ **다중 이미지** — 드래그앤드롭·클릭·URL로 여러 장 업로드, 썸네일에서 개별 제거. 카드엔 장수 배지, 상세엔 갤러리
- 📄 **상세 보기** — 요약, 핵심 포인트, 태그, 원본 메모, 사이트 링크
- ✏️ **편집/삭제**, 이미지 자동 압축 저장

## 파일 구성

|------|------|
| `index.html` | **앱 본체** (UI·Supabase·AI 호출) |
| `api/analyze.js` | AI 프록시 (Vercel 서버리스 함수, 키 숨김) |
| `README.md` | 이 문서 |
| `server.cjs`, `.claude/launch.json` | 미리보기용 로컬 서버 (선택) |

## 기술 구성

- **프런트엔드**: 단일 `index.html` (프레임워크 없음)
- **DB / 실시간**: Supabase (`services` 테이블, `postgres_changes` 구독)
- **AI**: `api/analyze.js` (Vercel 서버리스 함수)가 Claude를 대신 호출 — 키는 서버에만
- **배포**: Vercel. 코드 수정 후 `vercel deploy --prod`로 갱신

> **비밀 키 위치**
> - `ANTHROPIC_API_KEY` → Vercel 환경변수(Production). 코드·깃·브라우저에 노출 안 됨. AI 요금은 이 키 하나로 통합 청구.
> - Supabase `anon key`는 `index.html` 상단에 있으나 공개돼도 안전한 클라이언트용 키입니다.
